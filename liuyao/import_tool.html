錄 鴨鴨六爻

記錄轉檔小工具

把手上的卦記錄填進表格，就會自動整理成鴨鴨六爻能匯入的 JSON 檔。不用會寫程式。

放心：這個工具完全在你的瀏覽器裡執行，不連網、不上傳，你填的內容不會傳到任何地方。

產生的 JSON

下方會即時更新。填好後按「下載檔案」或「複製」，再到 App 的「記錄管理 → 選擇檔案匯入」匯入。

怎麼用

  1. 一筆記錄填一張卡片；同一次要匯入多筆，就按「新增一筆記錄」加卡片。
  2. 主卦是必填（點欄位會跳出六十四卦清單可選，也可以自己打字）。若這一卦有動爻、變出另一個卦，就填變卦；沒有動爻可留空。
  3. 其餘欄位（占者、所問、日期時間、記錄、標籤）都是選填，有就填、方便日後查找。
  4. 填好 → 下載檔案 → 傳到手機 → 在 App 用「選擇檔案匯入」選它即可。
